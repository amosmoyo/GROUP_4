About

CROMO MOBILE STRORE is a Simple MEANSTACK application, that is, its main areas are MongoDB,
Express, Angular and Nodejs.

Objective

The objective of the project is trying to see how we can create a fullstack web application,
thats is responsive, is functional and error free.

Areas of Coverage

Frontend

I used Angular2, angular material, angular cdk, bootstrap, css and html5.

Server

I used nodejs, express and Mongoose. installed some middleware,

Middleware APIs

cors
for resourse sharing (client-server communications).
body-parser/ Express.json()
To work with JSON data format
multer
To handle images
bcryptjs
To salt and hash passwords(encryption)
jsonwebtoken
Generating data tokens for authetification purpose.
passport and passport-jwt
To guard and protect routes.

Key Points

I used, cors , express.json() to work with json data format, multer to hundle images, bcryptjs to hash password,

Database

I used mongodb.
